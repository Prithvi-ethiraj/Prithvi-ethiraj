# 🛠️ System Health Monitor Script

A simple Python script that checks CPU, RAM, Disk usage, and logs it.  
Perfect for IT Support, L1 Engineers, or system admins.

## ✅ Features
- CPU usage monitoring  
- RAM usage  
- Disk space  
- System uptime  
- Logs output to `health_log.txt`

## 🚀 How to Run

```bash
pip install psutil
python monitor.py
